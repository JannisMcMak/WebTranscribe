# WebTranscribe

**WebTranscribe** is a tool for transcribing music inspired by [Transcribe!](https://www.seventhstring.com/xscribe/overview.html).
Everything works entirely in the browser with the help of WebAssembly.

## Features

- Audio playback
- Professional-grade playback rate control / pitch correction using [Rubberband](https://breakfastquay.com/rubberband/)
- Music analysis like beat/Rhythm detection and multi-pitch extraction powered by [Essentia](https://essentia.upf.edu)
